## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository, now with Docker support.

## Installation

Copy the content of the ``.env.example`` file to a new ``.env`` file (ignored).
Inside, you'll have to specify missing informations.

Then, ensure you have Docker and Docker Compose installed on your system. If not, [follow the Docker installation guide](https://docs.docker.com/get-docker/).

```bash
# Build and run the Docker container
$ sudo docker compose up
```

Once the Docker container is running, you need to initialize Prisma for database migrations and code generation:

```bash
# Generate Prisma artifacts
$ npx prisma generate
```
```bash
# Apply Prisma migrations
$ npx prisma migrate dev
```
## Running the app
With Docker running and Prisma set up, you can start the app:


```bash
# The app is already running in Docker, but you can check its status
$ sudo docker compose ps
```

## Test

To test the application, ensure Docker is running, then use the following commands:
```bash
# unit tests
$ npm run test
```
```bash
# end-to-end (e2e) tests
$ npm run test:e2e
```
```bash
# test coverage
$ npm run test:cov
```
